# Exercise 1 Example — Dijkstra

This example runs Dijkstra's algorithm on a small sample graph and shows the step-by-step distances and predecessors.

Files:

- `index.html` — this page.

How to use:
1. Press 'Run Dijkstra' to execute the example from a chosen source node.
2. Inspect the output to follow how distances are updated.
